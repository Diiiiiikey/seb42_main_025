# 소규모 커미션 홈페이지 [Mint Choco Tea]

마치 버블티처럼 작은 버블들이 모여서 맛있는 맛을 만들어내듯이, 작은 커미션들이 모여서 함께 좋은 커미션 홈페이지를 만들어 나가겠다는 의미가 담겨 있습니다.   
> "생소할 수도 있는 커미션이라는 부분을 모두 쉽게 접근할 수 있게 소규모 커미션들을 소개해드립니다."
